## Local Development

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open your browser and navigate to `http://localhost:8080`

## Deployment

This project is configured for automatic deployment on Railway:

- **Static Site**: Serves HTML, CSS, and JavaScript files
- **Node.js Server**: Custom server for static file serving
- **Health Check**: Configured for Railway's deployment monitoring
- **Environment**: Production-optimized settings
